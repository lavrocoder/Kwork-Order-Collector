# Kwork Order Collector

Расширение, для Chrome, которое добавляет на страницу заказа Kwork кнопку `Скачать raw.md`.

При нажатии этой кнопки вся переписка будет собрана в один файл и автоматически скачается

## Установка

1. Скачайте zip-архив со страницы [Releases](../../releases/latest) и распакуйте его в удобное место.
2. Откройте Chrome и перейдите на `chrome://extensions`.
3. Включите **Режим разработчика** (переключатель в правом верхнем углу).
4. Нажмите **Загрузить распакованное расширение** и выберите папку, в которую распаковали архив.
5. Расширение появится в списке и будет готово к работе.

## Использование

1. Войдите в аккаунт на [kwork.ru](https://kwork.ru).
2. Откройте страницу нужного заказа — URL должен иметь вид `kwork.ru/track?id=…`.
3. В правом нижнем углу появится кнопка **📥 Скачать raw.md**.
4. Нажмите её — расширение автоматически раскроет всю переписку и скачает файл `<id>_raw.md`.

## Структура выходного файла

Скачанный `.md`-файл содержит:

- **Мета-данные заказа** — номер, название услуги, покупатель, стоимость, дата оплаты, статус.
- **Переписка до заказа** — сообщения из раздела «Переписка, которая может относиться к заказу» (если есть).
- **Основная переписка** — все сообщения и системные события в хронологическом порядке. Для каждого сообщения указаны автор, дата, время и текст; цитаты оформлены в формате Markdown (`>`), прикреплённые файлы перечислены со ссылками.
- **Файлы заказа** — список всех вложений с именами и прямыми ссылками.
- **Отзывы** — текст отзыва покупателя и ответ исполнителя.
